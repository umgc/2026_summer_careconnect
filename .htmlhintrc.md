# HTMLHint Configuration Reference

Config file: `.htmlhintrc`
Enforced by: CI quality gate → HTMLHint step
Blocking: true

## Default Rules (HTMLHint built-in)

| Rule | Value | Purpose |
|------|-------|---------|
| `tagname-lowercase` | true | Tag names must be lowercase |
| `attr-lowercase` | true | Attribute names must be lowercase |
| `attr-value-double-quotes` | true | Attribute values must use double quotes |
| `doctype-first` | true | Doctype declaration must appear first |
| `tag-pair` | true | All opened tags must be closed |
| `spec-char-escape` | true | Special characters must be escaped |
| `id-unique` | true | id attributes must be unique per page |

## Added Rules (CareConnect)

| Rule | Value | Purpose |
|------|-------|---------|
| `doctype-html5` | true | Enforce modern HTML5 doctype |
| `src-not-empty` | true | src attributes must not be empty |
| `attr-no-duplication` | true | No duplicate attributes on an element |
| `alt-require` | true | All images must have alt text — ADA/accessibility compliance |
| `title-require` | true | All pages must have a title element — accessibility |
| `id-class-value` | "dash" | Enforce kebab-case for id and class names |
| `attr-unsafe-chars` | true | Flag unsafe characters in attribute values |
| `href-abs-or-rel` | false | Allow any href format (not enforced) |
| `inline-style-disabled` | false | Inline styles allowed (not enforced) |
| `style-disabled` | false | Style tags allowed (not enforced) |
| `script-disabled` | false | Script tags allowed (not enforced) |
| `space-tab-mixed-disabled` | "space" | Disallow mixed space/tab indentation |

## Change Control

Treat this file as governance configuration.
Modify rules carefully and commit changes via PR review.
Stricter rules should be enabled progressively as the codebase is cleaned up.
